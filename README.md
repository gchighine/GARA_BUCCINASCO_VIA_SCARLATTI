# Gara Buccinasco - Via Scarlatti

Questo repository è stato creato per organizzare e gestire tutta la documentazione e i materiali relativi alla partecipazione alla gara d'appalto per il progetto in Via Scarlatti, Buccinasco.

L'obiettivo è centralizzare le informazioni, facilitare la collaborazione tra i membri del team e mantenere una cronologia chiara dello sviluppo dell'offerta.

## Struttura del Repository

Il repository è organizzato in cartelle numerate per seguire un flusso di lavoro logico, dalla documentazione di gara iniziale fino alla consegna dell'offerta finale.

| Cartella | Contenuto |
| :--- | :--- |
| `01_DOCUMENTI_GARA` | Contiene tutti i documenti ufficiali forniti dall'ente appaltante (disciplinare, progetto, CSA, ecc.). **Questa cartella è il punto di partenza.** |
| `02_PROGETTI_RIFERIMENTO` | Una raccolta di progetti simili o di riferimento che possono essere usati come base o ispirazione per lo sviluppo della nostra offerta. |
| `03_CERTIFICAZIONI` | Tutte le certificazioni aziendali (ISO, Albo Gestori, ecc.) richieste per la partecipazione alla gara. |
| `04_OFFERTA_DA_SVILUPPARE` | **Il cuore del nostro lavoro.** Qui verranno sviluppate le bozze e la versione finale dell'offerta tecnica ed economica. |
| `05_AUTORIZZAZIONI_IMPIANTI` | Documentazione relativa alle autorizzazioni degli impianti di smaltimento, recupero e dei trasportatori. |

## Come Utilizzare Questo Repository

1.  **Clonare il Repository**: Per lavorare in locale, clona questo repository sul tuo computer.
2.  **Aggiungere i File**: Carica i documenti nelle rispettive cartelle. Ad esempio, il disciplinare di gara va in `01_DOCUMENTI_GARA/`.
3.  **Sviluppare l'Offerta**: Lavora sui documenti dell'offerta nella cartella `04_OFFERTA_DA_SVILUPPARE/Bozze_progressive/`. Una volta finalizzata, sposta la versione definitiva in `04_OFFERTA_DA_SVILUPPARE/Versione_finale/`.
4.  **Mantenere l'Ordine**: Non modificare la struttura delle cartelle principali. Se necessario, crea sottocartelle all'interno di quelle esistenti per mantenere l'organizzazione.
